#  Wallet Guardian (지갑지키미)

LG AI 캠프 - AI 기반 맞춤형 용돈 관리 및 지출 분석 어플리케이션입니다.

##  로컬 실행 방법 (PC에서 실행하기)

본 프로젝트는 React와 Vite 환경으로 구성되어 있습니다. 로컬 PC에서 실행하려면 아래의 단계를 따라주세요.

### 1단계: 필수 패키지 설치
터미널을 열고 터미널의 현재 위치가 프로젝트 폴더인지 확인한 뒤, 아래 명령어를 입력합니다.
(터미널 여는 방법: Window + R -> cmd 입력)
```bash
npm install
```

### 2단계: 로컬 서버 구동
패키지 설치가 완료되면, 아래 명령어를 통해 개발용 웹 서버를 실행합니다.
```bash
npm run dev
```

### 3단계: 브라우저에서 앱 확인
명령어를 실행하면 터미널에 아래와 비슷한 문구가 나타납니다.
```text
  ➜  Local:   http://localhost:5173/
```
해당 주소(`http://localhost:5173/`)를 키보드의 **`Ctrl` 키(Mac은 `Cmd`)를 누른 상태로 클릭**하거나, 브라우저 주소창에 복사하여 붙여넣으면 앱이 나타납니다.

**서버 종료 방법:** 실행 중인 터미널 창을 한 번 클릭하고 키보드의 `Ctrl + C`를 누르면 정상적으로 종료됩니다.

##  임시 온라인 테스트 (모바일 등으로 접속 시)

로컬 PC에서 띄운 서버를 잠시 스마트폰 등에서 접속해보고 싶을 때 사용하는 터널링 방법입니다.

1. 첫 번째 터미널에서 `npm run dev`를 실행 상태로 둡니다.
2. **새 터미널**을 하나 더 열어 아래 명령어를 입력합니다:
   ```bash
   npx localtunnel --port 5173
   ```
3. 터미널에 발급된 링크(`https://....loca.lt`)로 스마트폰에서 접속하여 테스트합니다. (로컬 서버를 끄면 이 링크도 자동으로 닫힙니다.)
